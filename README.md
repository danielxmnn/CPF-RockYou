# CPF-RockYou

## Overview

**CPF-RockYou** is a Python script designed to generate valid Brazilian CPFs (Cadastro de Pessoas FÃ­sicas). The script uses a custom algorithm to calculate the verification digits and produces a list of 1 billion valid CPFs, storing them in a file named `cpfs_validos.txt`.

## Features

- Calculates the verification digits for a given CPF.
- Generates 1 billion valid CPFs.
- Progress tracking using the `tqdm` library.

## Requirements

- Python 3.x
- `tqdm` library

## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/danielxmnn/CPF-RockYou.git
    ```
2. Navigate to the project directory:
    ```bash
    cd CPF-RockYou
    ```
3. Install the required library:
    ```bash
    pip install tqdm
    ```

Run the script to generate valid CPFs:
